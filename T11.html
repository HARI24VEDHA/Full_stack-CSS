CSS Positioning Techniques

Explore various CSS positioning techniques and their effects on layout. This page demonstrates the use of different positioning methods including static, relative, absolute, fixed, and sticky positioning.

1. Static Positioning

Static

Static positioning is the default positioning method. Elements are positioned according to the normal document flow and cannot be adjusted using the top, right, bottom, or left properties.

2. Relative Positioning

Relative

Relative positioning allows you to position an element relative to its original position. You can use the top, right, bottom, and left properties to adjust the element's position without affecting the layout of other elements.

3. Absolute Positioning

Absolute

Absolute positioning places an element relative to the nearest positioned ancestor (not static). If no positioned ancestor exists, it is placed relative to the initial containing block (the viewport).

4. Fixed Positioning

Fixed

Fixed positioning keeps an element fixed in the viewport, meaning it stays in the same position regardless of scrolling. This is often used for navigation bars or floating action buttons.

5. Sticky Positioning

Sticky

Sticky positioning is a hybrid between relative and fixed positioning. The element is treated as relative until it reaches a defined scroll position, at which point it sticks in place.
