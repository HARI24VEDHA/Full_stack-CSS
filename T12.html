Understanding Positioning

Static: The default positioning where elements flow naturally in the document. Elements are positioned according to the normal flow of the document, and top, right, bottom, and left properties have no effect.

Relative: Positioned relative to its normal position. It can be moved away from its normal flow using the top, right, bottom, and left properties, but it still occupies its original space in the document flow.

Absolute: Positioned relative to the nearest positioned ancestor (i.e., an element with a position other than static). It is removed from the normal document flow, meaning it will not affect the position of other elements.

Fixed: Positioned relative to the viewport. It stays in the same position even when the page is scrolled. Useful for elements like navigation bars that need to stay visible at all times.

Sticky: A hybrid of relative and fixed positioning. The element is treated as relative until it crosses a specified point in the viewport, at which point it is treated as fixed. Ideal for elements that need to stick to the top or bottom of the viewport when scrolled to.
