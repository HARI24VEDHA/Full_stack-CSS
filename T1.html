CSS Selectors and Advanced Styling

Types of Selectors with Different Styles

In this example, we will explore various types of CSS selectors used to style HTML elements. This paragraph is italicized and underlined using a class selector.

1. Element Selector

Element selectors target all instances of a specific element on a page. For example, all paragraphs are styled with the same font and color.

2. Class Selector

Class selectors allow you to apply styles to any element that shares a common class. This paragraph is italicized and underlined using a class selector.

3. ID Selector

This paragraph is styled uniquely using an ID selector. It is bold and stands out from other elements on the page.

4. Attribute Selector

Click here to visit Example. Links with a specific attribute (in this case, href starting with "https") are styled with an attribute selector and are italicized.

5. Pseudo-class and Pseudo-element

CSS pseudo-classes and pseudo-elements provide advanced styling options. Hover over the link to see the pseudo-class in action, and notice the large first letter at the start of each paragraph styled using a pseudo-element.
