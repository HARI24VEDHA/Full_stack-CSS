CSS Box Model Demonstration

Content Box Model

This element uses the content-box model. The width and height only apply to the content area. Padding and border are added outside of this area.

Border Box Model

This element uses the border-box model. The width and height include padding and border. This ensures the element stays the same size regardless of padding or border.
